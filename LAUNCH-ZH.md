## 0. 中文社区 vs HN：打法不同

| | HN / Reddit | 中文社区（你的主场） |
|---|---|---|
| 人群 | 陌生开发者，挑刺多 | 已关注你的粉丝 + AI 编程人群 |
| 入口 | 冷启动一条帖 | **你已有 14.7k+5.2k★ 仓库 + 公众号/抖音矩阵** |
| 网络 | 直连 GitHub/npm 没问题 | **GitHub/raw 访问慢**——文案必须给加速方案 |
| 调性 | 技术硬碰硬、忌营销腔 | 讲人话、讲场景、讲"省了什么事" |

**核心策略**：别从零冷启动。**用存量带增量**——你的公众号/抖音/视频号 + 两个高星仓库就是现成漏斗。先在自己渠道发，再外溢到垂直社区。

### ⚠️ 必带的加速提示（每条文案都要有）

国内直连 `raw.githubusercontent.com`（skillet 默认技能源）经常超时。给读者备好镜像：

```bash
# npm 包本身走 npm 官方/淘宝镜像都正常，无需特殊处理：
npx @jnmetacode/engram --version

# 只有 skillet 拉技能列表走 GitHub raw，慢的话用 jsDelivr 镜像：
npx @jnmetacode/skillet search pdf \
  --registry https://cdn.jsdelivr.net/gh/jnMetaCode/skillet@main/registry/index.json
```

（可在 `skillet.json` 里持久化这个 registry，文案里提一句即可。）

## 1. 微信公众号（你的主阵地，长文）

**标题候选**（公众号标题决定打开率，二选一 A/B）：
- A（痛点）：《AI 帮我写代码，但它转头就忘 —— 我做了 3 个本地小工具补上这个洞》
- B（场景）：《给你的 AI 装上"记忆 / 技能 / 眼睛"：三条命令，数据不出电脑》

**开头**（前 3 行决定是否读下去，讲场景不讲技术）：

> 你有没有这种感觉：用 Claude / Cursor 写代码，今天聊明白的事，明天它又问你一遍。
> 它没有记忆、不知道你团队的约定、你也看不到它每一步到底干了啥、花了多少 token。
>
> 我做了三个小工具补这三个洞，全部 **一条命令就能用、零依赖、数据不出你电脑**。

**正文结构**（建议）：
1. 三个洞 = 三个工具（每个配一张 GIF，仓库 `docs/demo.gif` 直接用）
   - 🧠 **记忆**（engram）：让 AI 记住你的笔记和决定，问它能带出处回答，**而且越用越准**（reinforce 那张 GIF 最抓人——分数肉眼可见上升）

   - 🔭 **眼睛**（tracelet）：实时看 AI 每步干了啥、花了多少钱
2. 为什么"本地 + 零依赖"重要：你的笔记/提示词/代码不上传任何人的云
3. 30 秒上手（贴上面的命令 + 加速提示）
4. 结尾引导：仓库地址 + "这是我 AI 全家桶的一块，配合 agency-agents-zh / superpowers-zh 用"

**结尾 CTA**：

## 6. 一句话差异化（被问"和 XX 比"时）

- 被问"和 Mem0/Dify 的记忆比"：它们是托管平台要联网；engram 是你能直接打开看的一个本地文件，离线也能用，还越用越准。
- 被问"为什么不用现成的 LangSmith"：那是生产监控仓库；tracelet 是你**写 agent 时**开在旁边的实时面板，零部署、关掉就没了。
- 被问"和我已有的 agency-agents-zh 什么关系"：那个给 AI **角色**，这个给 AI **记忆/技能/眼睛**，互补，一起用。
